
# MEDICAL RAG CHAT BOT

A Retrieval-Augmented Generation (RAG) based Medical Chatbot. With the base model as Llama 3.1 from Meta, the Retrieval system uses a medical document as corpus to generate context-rich output.
 
##  Model and dataset
* Model: Llama 3.1 8B, Use Groq API
* Embeddings from [hugging face](https://huggingface.co/sentence-transformers/all-MiniLM-L6-v2)
* Dataset: Gale Encyclopedia of Medicine Vol. 2 (N-S).pdf [Link](https://www.academia.edu/32752835/The_GALE_ENCYCLOPEDIA_of_MEDICINE_SECOND_EDITION)
